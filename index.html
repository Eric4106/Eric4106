Dotify

Description

Dotiy is a full-stack website application designed to be an altered clone of Spotify. Users can create an account with a username, password, and email. The app uses regular expressions to verify that they are valid entries. Once loged in, the user can create and name playlists, then add songs to that playlist. The app is run with Express and stores its data in SQL using Sqlite3.
